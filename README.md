# QSanguosha-v2-htms
Repo for QSanguosha-v2 lua extension 幻天漫杀 package.

## Installation
请以游戏启动文件所在目录为根目录，依次将压缩包中各个文件拖到对应文件夹，有可能需要覆盖
（本群569396837）
